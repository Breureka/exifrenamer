## Install

### Requirements

Python v2.7+

### Procedure

Check out the source:

    git clone git://github.com/wting/exifrenamer.git

Run:

    make install

## Uninstall

Delete the program directory:

    rm -fr ~/.exifrenamer/

Remove the following line from your `.bashrc`:

    export PATH=~/.exifrenamer/bin:"${PATH}"
